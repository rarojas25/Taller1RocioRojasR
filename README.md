# Taller - Procrastinación

## Descripción
Aplicación en Java para gestionar el registro y análisis de actividades diarias con el objetivo de evaluar la procrastinación de los usuarios.

## Funcionalidades principales
- Registro, modificación y eliminación de actividades.
- Cambio de contraseña de usuario.
- Análisis de la actividad más realizada en general y por usuario.
- Identificación del usuario con mayor procrastinación (más horas acumuladas).

## Uso
1. Clonar o descargar el repositorio.
2. Ejecutar la clase `Main` en el paquete `logica`.
3. Los datos se guardan en los archivos `Usuarios.txt` y `Registros.txt`.
4. Usar los menús por consola para interactuar con la aplicación.
